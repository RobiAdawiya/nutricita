# Nutricita 
adalah sistem rekomendasi cerdas yang dirancang untuk mengatasi tantangan pemilihan makanan optimal dengan menggabungkan analisis gizi mendalam dan analisis lokasi real-time. Menggunakan dataset yang kaya akan variabel kuantitatif (kalori, protein, lemak, karbohidrat), sistem ini mengimplementasikan algoritma rekomendasi untuk mempersonalisasi pilihan makanan sesuai kebutuhan diet pengguna, menunjukkan keahlian dalam bekerja dengan data kesehatan yang kompleks. Fitur unggulannya adalah Geospatial Proximity Analysis, di mana sistem tidak hanya merekomendasikan makanan yang seimbang secara gizi, tetapi juga mengintegrasikan lokasi pengguna saat ini untuk menemukan penyedia makanan terdekat, menunjukkan kemampuan praktis dalam analisis spasial dan pengembangan solusi teknologi yang berorientasi pada kenyamanan pengguna.
